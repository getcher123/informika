# Описание интерфейсов

| Страница / Раздел | Элемент UI | Описание / Поведение | Доступные роли |
|-------------------|------------|----------------------|----------------|
| Главная витрина | Верхний хедер | Меню: Банк инноваций, Конкурсы, заглушки «Медиасентр»/«Обучение», кнопка входа/аватар | Все роли, гость |
| Главная витрина | Блок «Как принять участие?» | Карточки с CTA «Стать участником», «Подать заявку» | Все / гость |
| Главная витрина | Промо-блок «Стать участником» | Переход к регистрации | Все / гость |
| Банк инноваций | Панель фильтров | Фильтры: статус, направление, автор, поиск, сортировка | Все роли |
| Банк инноваций | Карточка запроса | Направление, статус-чип, краткое описание, дедлайн, кнопка «Подробнее» | Все роли |
| Банк инноваций | Кнопка «Подать идею» | Доступна при активном запросе | Автор, участники |
| Карточка запроса | Разделы: проблема, критерии, сроки, вложения | Просмотр/редактирование данных запроса | Стейкхолдер (ред.), модератор (премодерация), прочие – просмотр |
| Карточка запроса | Счётчик идей | Количество поданных идей | Все |
| Создание идеи | Форма | Поля: заголовок, описание, эффект, вложения, согласие | Автор |
| Создание идеи | Кнопки | «Сохранить черновик», «Отправить на модерацию», «Отмена» | Автор |
| Карточка идеи | Шапка | Направление, статус, заголовок, ссылка на запрос, автор | Все |
| Карточка идеи | Вкладка «Описание» | Основной контент идеи | Все |
| Карточка идеи | Вкладка «История» / Журнал событий | Статусы, комментарии, действия | Все |
| Карточка идеи | Вкладка «Комментарии» | Комментарии стейкхолдера/координатора/участников | Участники, координатор, стейкхолдер |
| ЛК участника | Меню | «Мои идеи», «Заявки в команду», «Профиль» | Автор/участник |
| ЛК участника «Мои идеи» | Фильтры | Черновики, На модерации, На доработке, На рассмотрении, Одобрены, Отклонены, Архив | Автор |
| ЛК участника «Мои идеи» | Карточка идеи | Статус, комментарии, кнопки «Редактировать» (черновик / на доработке), «Подробнее» | Автор |
| ЛК участника «Заявки в команду» | Панель фильтров | Все / Входящие / Отправленные / Моя команда / Архив, «Сбросить» | Автор/участник |
| ЛК участника «Заявки в команду» | Блок «Моя команда» | Список участников, роли, действие «Удалить из команды» (для автора/координатора) | Автор, координатор |
| ЛК участника «Заявки в команду» | Карточка заявки | ФИО, комментарий, роль, дата, кнопки «Принять» / «Отклонить» / «Подробнее» | Автор, координатор |
| ЛК участника «Заявки в команду» | Командный воркфлоу | Схема задач (Новая, В работе, На проверке, Возврат, Готово) | Роли команды |
| ЛК участника «Заявки в команду» | Шаблоны задач | Перечень задач: Аналитика, План, Реализация, Тестирование, ПК/НТК, Итоговый отчёт | Координатор, автор |
| ЛК участника «Профиль» | Шапка профиля | Аватар, ФИО, организация/подразделение/должность, роли | Автор |
| ЛК участника «Профиль» | Контакты | Телефон, e-mail | Автор |
| ЛК участника «Профиль» | Смена пароля | Поля текущий/новый пароль, кнопка «Сохранить» | Автор |
| ЛК участника «Профиль» | Доп. поля по ролям | Поля для работника МОГ, студента, бизнеса и т.д. | Автор |
| CRM (модератор) | Список запросов | Табличный вид, статусы, фильтры | Модератор |
| CRM (модератор) | Карточка запроса / идеи | Одобрение/отклонение, комментарии | Модератор |
| ЛК стейкхолдера (Ф2) | Портфель идей | Грид «Одобрена/Готово к ПК/Готово к НТК», индикаторы прогресса | Стейкхолдер, координатор |
| ЛК стейкхолдера | Карточка одобренной идеи | Вкладки «Обзор», «Задачи», «Документы», «Команда», «Отчёты» | Стейкхолдер, координатор |
| ЛК стейкхолдера | Журнал событий | Изменения статусов, задач, документов | Стейкхолдер, координатор |
| ЛК стейкхолдера | Отчёты | Таблицы/диаграммы (статусы задач, активность команды) | Стейкхолдер, координатор, модератор |
| Bitrix24 | Канбан-задач | Колонки статусов, карточки с чек-листами, комментариями | Координатор, участники, стейкхолдер (просмотр) |
| Bitrix24 | Диск идеи | Папки: Требования, План, Прототипы, Тесты, ПК/НТК, Отчёты | Координатор, участники, стейкхолдер |
| Уведомления | e-mail / push / внутренние | Шаблоны события (статусы, задачи, документы) | Получатели по роли |
