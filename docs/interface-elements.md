# Описание интерфейсов (структурированный формат)

## Главная витрина
- **Верхний хедер:** пункты «Банк инноваций», «Конкурсы», заглушки «Медиасентр» и «Обучение», кнопка входа/аватар.
- **Блок «Как принять участие?»**: две карточки с CTA «Стать участником» и «Подать заявку».
- **Промо-блок «Стать участником»:** ведёт на регистрацию.

## Банк инноваций
- **Панель фильтров:** статус, направление, автор, поисковая строка, сортировка.
- **Карточка запроса:** направление, статус-чип, краткое описание, дедлайн, кнопка «Подробнее».
- **Кнопка «Подать идею»:** появляется для активных запросов и авторизованных участников.

## Карточка запроса
- **Основные разделы:** проблема, критерии, сроки, вложения.
- **Роли и действия:** стейкхолдер может редактировать, модератор — премодерация, остальные — только просмотр.
- **Служебные элементы:** счётчик идей, ссылки на вложения.

## Создание идеи
- **Поля формы:** заголовок, описание, ожидаемый эффект, вложения, согласие с правилами.
- **Доступные действия:** «Сохранить черновик», «Отправить на модерацию», «Отмена».

## Карточка идеи
- **Шапка:** направление, статус, заголовок, ссылка на запрос, автор.
- **Вкладка «Описание»:** основной контент идеи.
- **Вкладка «История»:** статусы, комментарии, действия.
- **Вкладка «Комментарии»:** обмен сообщениями между участником, координатором, стейкхолдером.

## Личный кабинет участника
- **Меню:** «Мои идеи», «Заявки в команду», «Профиль».
- **Вкладка «Мои идеи»:** фильтры по статусам (Черновик, На модерации, На доработке, На рассмотрении, Одобрена, Отклонена, Архив), карточки с быстрыми действиями.
- **Вкладка «Заявки в команду»:** фильтры (Все, Входящие, Отправленные, Моя команда, Архив), карточки заявок, кнопки «Принять/Отклонить/Подробнее», блок «Моя команда».
- **Задачи:** кнопка «Открыть задачи в Bitrix24» (схема ведётся в CRM).
- **Профиль:** аватар, ФИО, организация, роли, контакты, смена пароля, дополнительные поля по роли.

## CRM модератора
- **Список запросов:** таблица со статусами и фильтрами.
- **Карточка запроса/идеи:** действия по одобрению, отклонению, комментарии.

## Личный кабинет стейкхолдера (Фаза 2)
- **Портфель идей:** грид с состояниями «Одобрена», «Готово к ПК», «Готово к НТК».
- **Карточка одобренной идеи:** раздел «Обзор» внутри ЛК; вкладки «Задачи», «Документы», «Отчёты» открывают соответствующие разделы Bitrix24.
- **Журнал событий:** фиксирует статусы и ключевые решения; детализация по задачам и документам доступна в Bitrix24.
- **Отчёты:** кнопка перехода к отчётам CRM (статусы задач, активность команды).

## Bitrix24 (внешние разделы)
- **Канбан-задач:** статусы задач, чек-листы, комментарии.
- **Диск идеи:** папки «Требования», «План», «Прототипы», «Тесты», «ПК/НТК», «Отчёты».

## Уведомления
- **Каналы:** e-mail, push, внутренние сообщения.
- **События:** изменения статуса идеи, решения стейкхолдера, заявки в команду, обновления задач и документов (см. `docs/notification-table.md`).
