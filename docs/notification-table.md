# Уведомления (триггер → получатель)

|---------|------------|--------------------------|
| Участник отправил идею на модерацию | Модератор, стейкхолдер | Идеи |
| Модератор вернул идею на доработку | Автор идеи | Идеи |
| Модератор сменил статус идеи на «На рассмотрении» | Автор идеи | Идеи |
| Стейкхолдер принял решение («Одобрена» / «Отклонена») | Автор идеи | Идеи |
| Стейкхолдер отправил идею на доработку | Автор идеи | Идеи |
| Идея переведена в статус «Одобрена» | Участники команды | Идеи |
| Участник подал заявку в команду | Автор идеи, координатор | Команда |
| Заявка в команду одобрена/отклонена | Автор заявки | Команда |
| Освобождена обязательная роль в команде | Автор идеи | Команда |
| Добавлен новый документ в идее (Bitrix Диск) | Автор идеи, участники команды | Bitrix |
| Просрочена задача Bitrix | Исполнитель | Bitrix |
| Задача завершена | Автор идеи | Bitrix |
| Дедлайн запроса завершён | Авторы идей по запросу | Системные |
| Идея переведена в «Архив» | Команда идеи | Идеи |
| Пользователь зарегистрировался на мероприятие | Пользователь | События |
| Напоминание за 24 часа до начала мероприятия | Пользователь (зарегистрированный) | События |
| Напоминание за 2 часа до начала мероприятия | Пользователь (зарегистрированный) | События |

## Стейкхолдер

| Триггер | Получатель | Тип уведомления (фильтр) |
|---------|------------|--------------------------|
| Участник отправил идею на модерацию | Стейкхолдер | Идеи |
| Модератор сменил статус идеи на «На рассмотрении» | Стейкхолдер | Идеи |
| Стейкхолдер принял решение («Одобрена» / «Отклонена») | — (инициатор) | — |
| Идея переведена в статус «Одобрена» | Стейкхолдер | Идеи |
| Освобождена обязательная роль в команде | Стейкхолдер, координатор | Команда |
| Участник подал заявку в команду | Координатор, стейкхолдер (копия) | Команда |
| Заявка в команду одобрена/отклонена | — (решение в Bitrix24) | — |
| Добавлен новый документ в идее (Bitrix Диск) | Стейкхолдер, координатор | Bitrix |
| Просрочена задача Bitrix | Координатор, стейкхолдер (копия) | Bitrix |
| Задача завершена | Координатор | Bitrix |
| Дедлайн запроса завершён | Стейкхолдер, координатор | Системные |
| Идея переведена в «Архив» | Стейкхолдер, координатор | Идеи |

Уведомления по событиям (регистрация, напоминания, отмена, материалы) отправляются по e‑mail и/или отображаются в ленте уведомлений портала (тип `События`) по правилам Ф3. События подготовки/готовности к ПК и НТК обрабатываются напрямую в Bitrix24 и не транслируются в пользовательские уведомления портала. Управление заявками и ролями команды выполняется в Bitrix24; уведомления на витрине ведут к карточке идеи или соответствующей задаче Bitrix24 (только просмотр статуса).
