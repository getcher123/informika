# Уведомления (триггер → получатель)

| Триггер | Получатель |
|---------|------------|
| Участник отправил идею на модерацию | Модератор, стейкхолдер |
| Модератор вернул идею на доработку | Автор идеи |
| Модератор сменил статус идеи на «На рассмотрении» | Автор идеи, стейкхолдер |
| Стейкхолдер принял решение («Одобрена» / «Отклонена») | Автор идеи |
| Стейкхолдер отправил идею на доработку | Автор идеи |
| Дедлайн запроса завершён | Подписчики запроса (стейкхолдер, координатор, участники) |
| Идея переведена в статус «Одобрена» | Координатор, участники команды, стейкхолдер |
| Освобождена обязательная роль в команде | Координатор, автор идеи |
| Участник подал заявку в команду | Автор идеи, координатор |
| Заявка в команду принята/отклонена | Автор заявки |
| Добавлен новый документ в идее (Bitrix Диск) | Стейкхолдер, координатор, исполнитель по задаче |
| Просрочена задача Bitrix | Исполнитель, координатор |
| Задача завершена | Координатор, автор идеи |
| Идея переведена в «Архив» | Команда идеи (исполнители, координатор), стейкхолдер |

События подготовки/готовности к ПК и НТК обрабатываются напрямую в Bitrix24 и не транслируются в пользовательские уведомления портала.
