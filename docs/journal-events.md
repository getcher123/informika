# Журналы событий идей и запросов

## Журнал идеи

| Событие | Инициатор | Описание | Пояснение |
|---------|-----------|----------|-----------|
| Черновик создан | Автор идеи | Идея впервые сохранена в черновик | Фиксирует дату создания и автора |
| Отправка на модерацию | Автор идеи | Идея передана модератору, описания и файлы зафиксированы | История фиксирует версию и вложения |
| Возврат на доработку | Модератор | Идея возвращена с перечнем замечаний | Причина возврата обязательна, отправляется автору |
| Публикация для стейкхолдера | Модератор | Идея переведена в статус «На рассмотрении» | В журнале фиксируется модератор и комментарий |
| Отправлено на доработку стейкхолдером | Стейкхолдер | Идея возвращена автору, указана причина | Событие запускает уведомление и задачу на канбан-доске |
| Одобрена стейкхолдером | Стейкхолдер | Идея перешла в «Одобрена» и подготовлена к запуску | Запускает автоматизацию Bitrix24 |
| Отклонена стейкхолдером | Стейкхолдер | Идея окончательно отклонена | Причина обязательна, уведомление автору |
| Переведена в «Архив» | Стейкхолдер / Координатор | Идея завершена, материалы переданы во внешний проект | Журнал фиксирует итог и ссылку на отчёт |
| Назначение участника на роль | Координатор | Пользователь получил роль в команде идеи | Отражает роль и дату |
| Исключение участника / освобождение роли | Координатор | Участник снят с роли | Фиксирует причину и инициатора |
| Принятие приглашения | Участник | Пользователь подтвердил участие | Указывает роль и дату присоединения |
| Завершение проверки готовности к ПК/НТК | Координатор / Стейкхолдер | Отмечено выполнение контрольного чек-листа | Сопровождается ссылкой на задачу канбан-доски |

## Журнал запроса

| Событие | Инициатор | Описание | Пояснение |
|---------|-----------|----------|-----------|
| Черновик создан | Стейкхолдер | Создан новый запрос, заполнены базовые поля | Фиксируются реквизиты автора |
| Публикация (На сборе идей) | Стейкхолдер / Модератор | Запрос опубликован и доступен участникам | Фиксируется дата старта и модератор |
| Закрытие сбора идей | Автоматически / Стейкхолдер | Приём идей завершён (по дедлайну или вручную) | Содержит причину закрытия |
| Поступление новой идеи | Автор идеи | В запрос добавлена новая идея | Журнал отображает ссылку на идею и автора |
| Решение по идее: Одобрена | Стейкхолдер | Идея переведена в одобренные | Указывается идентификатор идеи и результат |
| Решение по идее: На доработку | Стейкхолдер | Идея возвращена автору, причина обязательна | Синхронизируется с журналом идеи |
| Решение по идее: Отклонена | Стейкхолдер | Идея отклонена | Фиксируется причина |
| Обновление материалов запроса | Стейкхолдер / Координатор | Изменены критерии, сроки или вложения | Журнал фиксирует что обновлено и кем |
