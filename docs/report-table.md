# Отчётность по ролям (Фаза 3)

| Роль | Отчёт / дашборд | Источник | Содержание / назначение | Периодичность и доступ |
|------|-----------------|----------|-------------------------|------------------------|
| Стейкхолдер | «Портфель одобренных идей» | Портал (ЛК стейкхолдера, Ф2) | Статус каждой одобренной идеи (Одобрена / Готово к ПК / Готово к НТК), ключевые дедлайны, риск-флаги, количество просроченных задач. | Доступ 24/7; обновление в реальном времени по данным Bitrix24. |
| Стейкхолдер | «Статусы задач по идее» | Bitrix24 → задачи (канбан) | Распределение задач по колонкам, чек-листы, просрочки, комментарии. | По запросу; доступ только к идеям, где стейкхолдер назначен владельцем. |
| Координатор | «Активность команды» | Bitrix24 → стандартный отчёт | Кол-во созданных/закрытых задач, сотрудники с просрочками, доля выполненных чек-листов. | Ежедневно; расширенный фильтр по идеям/направлениям. |
| Координатор | «Контроль ролей команды» | Портал (карточка идеи) | Заполненность обязательных ролей, история назначений/исключений участников. | При каждом изменении состава; уведомления при пустых слотах. |
| Модератор | «Журнал переходов статусов» | Портал (админ-панель) | Список переводов статусов идей и запросов, кто инициатор, комментарии. | В реальном времени; экспорт CSV для регуляторной отчётности. |
| Автор идеи / участник | «Мои идеи и заявки» | Портал (ЛК участника) | Текущий статус каждой идеи, замечания модератора, история заявок в команду. | Постоянно доступно; push/e-mail при обновлениях. |
| Автор идеи / участник | «Назначенные задачи» | Bitrix24 → личный список задач | Все задачи по идее, где участник исполнитель или наблюдатель. | Встроенные напоминания Bitrix24; фильтры по срокам. |
| Эксперт / Ментор / Трекер | «Мониторинг сопровождения» | Bitrix24 → задачи/отчёты | Задачи с участием роли, комментарии, чек-листы, статус подготовки к ПК/НТК. | По мере назначения; доступ ограничен идеями роли. |
| Руководство МОГ / PMO | «Консолидированный отчёт» | Экспорт из Bitrix24 + портал | Количество одобренных идей по направлениям, процент готовности, статус прохождения ПК/НТК. | По запросу (неделя/месяц); формирует координатор, выкладывает в `/Отчёты`. |
| Служба качества / аудит | «Архив идей» | Портал (статус «Архив») | Перечень идей, переведённых в архив, итоговые документы, причины решений. | По запросу; доступ только пользователям с правами аудита. |
