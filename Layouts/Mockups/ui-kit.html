Текстовые списки

`.feature-list` используется для перечисления преимуществ, шагов и особенностей внутри карточек. Класс управляет отступами и декоративными маркерами, поэтому его можно применять на любой странице.

Ненумерованные списки

  • Маркированные элементы получают единый отступ слева
  • Декоративный индикатор цвета `--color-secondary`
  • Гибкая высота — блок растягивается по содержимому

Общие требования верстке

Типографика

  • H1/H2 + .heading-page — главные заголовки страниц.
  • H2/H3 + .heading-section — карточки и блоки.
  • .heading-subtitle и .text-lead идут сразу под заголовком.
  • Любой текст размещаем в p, ul, ol; без «голых» div.

Каркас страниц

  • .body--gradient + .main--with-gradient для всех внутренних экранов.
  • Личный кабинет: .sidebar-layout = меню (.sidebar) + контент (.content-card).
  • Детальная страница: .detail-layout = основная колонка + .detail-sidebar.
  • .section и .card поддерживают вертикальный ритм (spacing-xl → spacing-md).

Компоненты и классы

  • Статусы всегда через .status-indicator--*; направления — .badge--*.
  • Списки преимуществ и шагов — .feature-list / .feature-list--ordered.
  • Поиск — .search; фильтры — .filters + .filter-pill.
  • Документы в карточках выводим только через .document-list.
