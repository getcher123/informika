Вертикальные отступы

Используйте готовые utility-классы, чтобы секции и элементы автоматически сохраняли согласованный вертикальный ритм.

Секции

Соседние .section разделяются отступом var(--spacing-2xl) — достаточно добавить класс к блоку.

<section class="section">...</section>
<section class="section">...</section>

Поиск и фильтры

.search → spacing-lg, .filters / .filters-section → spacing-xl. Карточки всегда находятся на одинаковом расстоянии от фильтров.

<div class="search">...</div>
<div class="filters filters-section">...</div>

.back-link автоматически оставляет spacing-lg до следующего блока.

</div>

Табы и фильтры

Табы

Контейнер `.tabs` + кнопки `.tabs__item` создают адаптивные вкладки. Добавьте `data-tabs` и `data-tab` при необходимости JS-переключения.

<div class="tabs" data-tabs>\n <button class="tabs__item tabs__item--active" data-tab="first">Вкладка 1</button>\n <button class="tabs__item" data-tab="second">Вкладка 2</button>\n</div>

Header (Верхнее меню)

Общие классы:

  • .header - основной контейнер
  • .header__container - внутренний контейнер
  • .header__logo - логотип
  • .header__nav - навигация
  • .nav__list - список навигации
  • .nav__link - ссылка навигации
  • .nav__link--active - активная ссылка
  • .header__actions - правая часть с действиями
  • .header__user-btn - кнопка пользователя
  • .header__user-btn--logged - для авторизованного пользователя
  • .header__user-initials - инициалы пользователя

Sidebar (Боковая панель)

Общие классы:

  • .sidebar - основной контейнер
  • .sidebar-layout__sidebar - позиционирование
  • .sidebar__nav - навигация
  • .sidebar__link - ссылка меню
  • .sidebar__link--active - активная ссылка
  • .sidebar__icon - иконка пункта меню
  • .sidebar__text - текст пункта меню
  • .sidebar__badge - бейдж с количеством

Классы

.form-container - основной контейнер (max-width: 600px)
.form-container--large - большой контейнер (max-width: 800px)
.form-container--small - маленький контейнер (max-width: 480px)
.form-container__header - шапка формы
.form-container__title - заголовок формы
.form-container__subtitle - подзаголовок формы
.form-container__body - тело формы
.form-container__footer - футер формы
